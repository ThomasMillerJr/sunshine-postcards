# Sunshine Postcards ŌĆö UI Design Spec

## Aesthetic Direction: Sunny & Clean

Warm, approachable, restrained. The postcards themselves provide the color ŌĆö the UI stays out of the way.

### Color Palette (3 colors only)

| Token | Value | Usage |
|-------|-------|-------|
| `--sun-gold` | `#F7B733` | Primary accent, active filters, golden highlights |
| `--sun-gold-light` | `#FFF4D6` | Active chip backgrounds, subtle highlights |
| `--coral` | `#E8634A` | Values, active nav, logo text, CTA hover |
| `--coral-light` | `#FFF0EB` | Hover states |
| `--body-bg` | `#FFFCF5` | Page background (warm cream) |
| `--card-bg` | `#FFFFFF` | Cards, nav |
| `--card-border` | `#FFF0D4` | Card borders, dividers |
| `--text-primary` | `#2D2A26` | Headings, titles |
| `--text-secondary` | `#8A8278` | Body text |
| `--text-muted` | `#B8B0A4` | Labels, meta, counts |

### Typography

| Role | Font | Weight |
|------|------|--------|
| Headings | DM Serif Display | 400 |
| Body / UI | DM Sans | 300, 500, 700 |

### Badges (muted, not bright)

| Status | Background | Text Color |
|--------|-----------|------------|
| Listed | `#FFF4D6` (sun-gold-light) | `#8A6A10` |
| Sold | `#E8F5E9` | `#2E7D32` |
| Unlisted | `#F5F0EA` | `#B8B0A4` |

## Design Principles

- **Restrained palette** ŌĆö only gold, coral, and warm neutrals. No rainbow.
- **Photos provide color** ŌĆö UI chrome is neutral warm gray/cream
- **Large touch targets** ŌĆö designed for a non-technical user
- **Cards not clutter** ŌĆö generous whitespace, clear hierarchy
